# Project M10 - AI Receptionist (Ajman Shop)

## Overview

AI Receptionist system for a retail shop in Ajman. Handles customer inquiries 24/7 via WhatsApp and web chat, captures leads, automates follow-ups, and sends daily reports to the shop owner.

## Workflows

| # | Workflow | File | Purpose |
|---|---------|------|---------|
| 1 | AI Receptionist | `m10-ai-receptionist.json` | Core chatbot - answers FAQs, takes orders, captures leads (Arabic + English) |
| 2 | WhatsApp Connector | `m10-whatsapp-connector.json` | Routes WhatsApp messages to the AI receptionist and sends replies back |
| 3 | Business Hours Router | `m10-business-hours-router.json` | Checks UAE business hours, routes to AI or sends after-hours auto-reply |
| 4 | Lead Follow-Up | `m10-lead-follow-up.json` | Auto follow-up with leads (2h for new, 24h for contacted, max 3 attempts) |
| 5 | Daily Summary | `m10-daily-summary.json` | Sends daily stats to owner via Telegram at 9 PM UAE time |
| 6 | Google Review Request | `m10-google-review-request.json` | Asks converted customers for a Google review 1-7 days after purchase |

## Architecture

```
Customer (WhatsApp/Web)
        │
        ▼
┌─────────────────────┐
│  Business Hours      │
│  Router              │──── After Hours ──→ Auto-reply + Log
└─────────┬───────────┘
          │ Open
          ▼
┌─────────────────────┐     ┌──────────────────┐
│  WhatsApp Connector  │────▶│  AI Receptionist  │
│  (Parse + Reply)     │◀────│  (GPT-4o-mini)   │
└──────────────────────┘     └────────┬─────────┘
                                      │
                            ┌─────────┼─────────┐
                            ▼         ▼         ▼
                       Save Lead   Notify    Follow-Up
                       (Sheets)    Owner     Scheduler
                                 (Telegram)     │
                                                ▼
                                          Daily Summary
                                          + Review Requests
```

## Setup Checklist

### 1. Replace Placeholders
In ALL workflow files, replace:
- `[SHOP NAME]` - The actual shop name
- `[FULL ADDRESS]` - Shop address in Ajman
- `[PHONE NUMBER]` - Shop phone
- `[WHATSAPP NUMBER]` - WhatsApp business number
- `YOUR_SPREADSHEET_ID` - Google Sheets ID for lead tracking
- `OWNER_TELEGRAM_CHAT_ID` - Owner's Telegram chat ID
- `YOUR_PHONE_NUMBER_ID` - WhatsApp Cloud API phone number ID
- `YOUR_GOOGLE_REVIEW_LINK` - Google Maps review link
- `REPLACE_WITH_YOUR_OPENAI_CREDENTIAL_ID` - n8n OpenAI credential ID

### 2. Google Sheets Structure
Create a Google Sheet with these tabs:

**Tab: Leads**
| Column | Description |
|--------|-------------|
| name | Customer name |
| phone | Phone number |
| inquiry | What they asked about |
| type | order / inquiry / complaint / booking |
| language | en / ar |
| timestamp | ISO timestamp |
| status | new / contacted / converted |
| source | AI Receptionist |
| follow_up_count | Number of follow-ups sent |
| last_follow_up | Last follow-up timestamp |
| converted_at | Conversion timestamp |
| review_requested | yes / (blank) |
| review_requested_at | Review request timestamp |

**Tab: After Hours Messages**
| Column | Description |
|--------|-------------|
| phone | Phone number |
| name | Customer name |
| message | Their message |
| received_at | Timestamp |
| status | pending_reply / replied |

### 3. n8n Credentials Needed
- OpenAI API (for GPT-4o-mini)
- Google Sheets OAuth2
- Telegram Bot API
- WhatsApp Cloud API (HTTP Header Auth with Bearer token)

### 4. Import Order
Import workflows into n8n in this order:
1. `m10-ai-receptionist.json` (core - must be first)
2. `m10-business-hours-router.json`
3. `m10-whatsapp-connector.json`
4. `m10-lead-follow-up.json`
5. `m10-daily-summary.json`
6. `m10-google-review-request.json`

### 5. WhatsApp Business API Setup
- Create Meta Business account
- Set up WhatsApp Business API via Meta Cloud API
- Configure webhook URL to point to `m10-whatsapp-incoming`
- Verify the webhook with Meta's verification challenge

## Business Hours (UAE)
- Saturday - Thursday: 9 AM - 10 PM
- Friday: 2 PM - 10 PM (after Jummah prayer)
- Adjust in `m10-business-hours-router.json` if the shop has different hours

## Follow-Up Schedule
- New leads: First follow-up after 2 hours
- Contacted leads: Next follow-up after 24 hours
- Maximum 3 follow-ups per lead
- Messages generated by AI in customer's preferred language
